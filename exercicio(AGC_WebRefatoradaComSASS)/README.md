# Agência Criativa Web

Estrutura refatorada em SASS com partials, variáveis, mixins, aninhamento e BEM.

## Estrutura
- `scss/` → código-fonte SASS
- `css/style.css` → CSS compilado
- `index.html` → página principal
- `script.js` → comportamento do cabeçalho fixo

## Compilação
```bash
npm install
npm run build:sass
```

Os arquivos de imagem não foram incluídos neste pacote porque não estavam disponíveis na entrega original.
